# PureShot Site

Public support, privacy, and App Store review site for PureShot: Process Zero Cam.

This repository intentionally contains only static website content:
- `index.html`
- `privacy/index.html`
- `support/index.html`
- `terms/index.html`
- `assets/`

It does not contain application source code, build scripts, secrets, private review notes, or internal launch documents.
