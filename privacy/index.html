Privacy Policy

Zero data collection.

PureShot is built as a local camera utility. The free launch app does not collect personal data, does not track users, and does not upload captured media to a cloud service.

Data collection

PureShot does not collect data from this app. It does not use third-party analytics SDKs, advertising SDKs, tracking SDKs, cookies, or account systems.

Local processing

Photos, videos, and capture metadata stay on the device unless the user explicitly exports or shares media through iOS system flows.

Permissions

  • Camera access is used for the live viewfinder and capture.
  • Microphone access is requested only for video capture flows that need audio.
  • Photos access is used only when the user saves or exports captured media.
